This project is a personal portfolio showing some of my past projects. 
It uses Python and Tkinter to create a central Hub where some of my favorite projects can be ran.
It also uses a function that uses subprocess, os, and sys to be able to run on mac and linux.


## How to use
***
1. Make sure python 3.10 is installed.
2. Run project.
3. Use menu to decide what to do.
4. Exit with X, or by pressing one of the numbers when launched.


## Details on Project features
***
- Detects the OS and opens each project in its own terminal window
- GUI stays open while the project is ran independently
- Built with Tkinter, requires python 3.10+
